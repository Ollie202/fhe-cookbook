# FHE operations

All operations live on the `FHE` library imported from `@fhevm/solidity/lib/FHE.sol`. They take encrypted handles and return new encrypted handles. None of them produce plaintext.

## Arithmetic

```solidity
FHE.add(a, b)   // a + b mod 2^N (no overflow check — see AP-011)
FHE.sub(a, b)   // a - b mod 2^N
FHE.mul(a, b)   // a * b mod 2^N
FHE.neg(a)      // -a (two's complement)
FHE.min(a, b)
FHE.max(a, b)
FHE.div(enc, plainConst)  // ⚠ divisor MUST be plaintext
FHE.rem(enc, plainConst)  // ⚠ divisor MUST be plaintext
```

## Bitwise

```solidity
FHE.and(a, b)
FHE.or(a, b)
FHE.xor(a, b)
FHE.not(a)
FHE.shl(a, n)   // n can be plaintext or encrypted
FHE.shr(a, n)
FHE.rotl(a, n)
FHE.rotr(a, n)
```

## Comparison — return `ebool`

```solidity
FHE.eq(a, b)   // ==
FHE.ne(a, b)   // !=
FHE.lt(a, b)   // <
FHE.le(a, b)   // <=
FHE.gt(a, b)   // >
FHE.ge(a, b)   // >=
```

You **cannot** use these in a Solidity `if` directly. Pass to `FHE.select` or to `FHE.and/or/not` for compound logic.

## Selection (the replacement for `if`)

```solidity
euint64 winner = FHE.select(cond /* ebool */, ifTrue, ifFalse);
```

Always branchless. The cost is *one full ciphertext op* even though you're "picking" — there is no short-circuit.

## Casting

```solidity
FHE.asEbool(x)
FHE.asEuint8(x) ... FHE.asEuint256(x)
FHE.asEaddress(x)
```

Both encrypted-to-encrypted (width change) and plaintext-to-encrypted (e.g. `FHE.asEuint64(uint64(0))`).

## Encrypted constants from plaintext

```solidity
euint64 zero = FHE.asEuint64(0);     // ciphertext of 0
ebool t = FHE.asEbool(true);
```

These are cheap; they're encryption-of-known-value (trivial encryption).

## Random

```solidity
euint32 r = FHE.randEuint32();
euint64 r64 = FHE.randEuint64();
```

Generated by the coprocessor; suitable for in-contract randomness without needing a VRF. **Do not use `block.timestamp` or `blockhash` to seed FHE values** — they're public.

## Input ingestion

```solidity
euint64 x = FHE.fromExternal(externalHandle, inputProof);
```

Verifies the proof, registers the handle for the calling contract. Always the first thing you do with user input.

## Initialisation check

```solidity
if (!FHE.isInitialized(handle)) { ... } // OK — this is plaintext metadata, not the value
```

## Cost intuition

Approximate ranking, cheapest first: `select ≈ add ≈ sub` < `cmp` < `mul` < `div(plain)` ≪ encrypted-divisor anything (don't). Bit width multiplies all of them. When you have a hot path, prefer fewer wider ops over many narrower ones — but right-size the type for the value range first.
